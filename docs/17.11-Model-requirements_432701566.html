Deep In Motion : 17.11 Model requirements

Created by Felix Tempel on 02.07.2025

While not tabular, the model’s biomechanical inputs are bounded through normalization and augmentation.

Spatial bounds (skeleton scale), temporal windows (5s), and augmentation ranges (rotation ±45°, scaling 0.7–1.3) define operational input space.

CAM (eFigure 3) helps localize body regions contributing to predictions.
