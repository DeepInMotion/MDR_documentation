Welcome to the documentation and MDR page for  the Deep In Motion project. Here you will find extensive documentation for the MDR readiness. We base our reporting on the work done by the authors:

The tables and guideline is cloned from:
GitHub - johner-institut/ai-guideline: Guideline for AI based Medical Devices

and is used non-commercial in order to document the project, as far as possible.
