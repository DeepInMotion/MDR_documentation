25.1

The manufacturer has created a Post-Market Surveillance (PMS) Plan.

25.3

The manufacturer has specified in the PMS plan the quality criteria and threshold values that it considers necessary for handling of in particular a re-evaluation of the risk-benefit analysis.


2017/745/EU, Chapter VII, 2017/746/EU, Chapter VII

25.4

The manufacturer has analyzed when determining these threshold values which feedback loops the threshold values can influence35.

This analysis also serves as a measure against the above risk through "Performance Prediction".
