Deep In Motion : 12.5 Value detection

Created by Felix Tempel, last updated by Daniel Groos on 19.08.2025

Requirements

  • This procedure specifies how values are detected and processed that have been collected with various measurement methods.

Summary

All data was collected via a standardized motion-tracking algorithm applied to the GMA videos.

No mixed modality or sensor types were used.

This reduces variability and avoids the issue of multi-source measurement inconsistencies.

In the future, the incorporation of smartphones video recordings is planned
