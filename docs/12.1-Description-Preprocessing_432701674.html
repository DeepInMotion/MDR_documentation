Deep In Motion : 12.1 Description Preprocessing

Created by Felix Tempel on 03.07.2025

The skeleton sequences were extracted from raw video using the EfficientPoseIII motion capture algorithm (Groos et al.).

Time series were resampled to 30 Hz and smoothed using a 5-point temporal median filter.

Skeleton coordinates were normalized based on pelvis location and trunk length to standardize scale and remove variability.
