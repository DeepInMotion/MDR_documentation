Deep In Motion : 16.2 Quality measure

Created by Felix Tempel on 17.06.2025

For each model under consideration, the following classification metrics were computed using a confusion matrix:

  • True Positives (TP)

  • False Positives (FP)

  • True Negatives (TN)

  • False Negatives (FN)

From these values, the following performance metrics were calculated:

  • Sensitivity (Recall)

  • Specificity

  • Accuracy

  • 95% Confidence Intervals (CI) using the Clopper-Pearson method
