Summary

For each model under consideration, the following classification metrics were computed using a confusion matrix based on setting the classification threshold at the sensitivity of GMA (see Methods section in reference 1 for details):

  • True Positives (TP)

  • False Positives (FP)

  • True Negatives (TN)

  • False Negatives (FN)

From these values, the following performance metrics were calculated:

  • Sensitivity (Recall)

  • Specificity

  • Accuracy

  • 95% Confidence Intervals (CI) using the Clopper-Pearson method
