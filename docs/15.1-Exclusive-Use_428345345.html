Deep In Motion : 15.1 Exclusive Use

Created by Felix Tempel, last updated by Daniel Groos on 17.06.2025
  • The model training, tuning, and selection process was confined exclusively to the development set, which consisted of 75% of the full dataset (N = 557 infants).
  • Within this development set, 7-fold cross-validation was applied, ensuring balanced representation of classes (Cerebral Palsy [CP] vs. No CP) and recording sites across folds.
  • For each fold, ten model variants were searched through the architecture search and trained from scratch. This was repeated across all folds. 
  • Hyperparameter tuning and model architecture selection were done based solely on performance in the cross-validation folds.
  • The external validation set (25%) was never used during model development or selection, and was solely for reporting the Test set performance.
    • The test set performance is the median prediction of all 70 models.
  • Performance improvements during training were monitored using validation AUC (Area Under the ROC Curve).
  • An early stopping mechanism was used to avoid overfitting, halting training when validation performance plateaued.
