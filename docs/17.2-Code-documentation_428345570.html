Deep In Motion : 17.2 Code documentation

Created by Felix Tempel, last updated by Pål Haugen on 18.08.2025

Code within the repository adheres to the following standards:

Each Python module includes:

  • Inline documentation for the inputs/outputs and purpose 

  • Explanation of expected tensor shapes

  • Function and class docstrings

The YAML configs include inline comments for each hyperparameter.

Documentation is maintained in markdown, and the landing page is supported with a readme.md to facilitate accessibility.

For the Deep in Motion Explorer a Sphinx documentation is set up under: https://github.com/DeepInMotion/DIME/tree/main/docs (needs access permission!)
