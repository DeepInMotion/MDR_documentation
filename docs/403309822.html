The intended use also suggests what the goal of machine learning techniques is.

Definition

The goal of the machine learning technique is early prediction of CP in human infants.

The two types of machine learning techniques is used to obtain the overall goal:

  1. Computer vision models for automatic identification/tracking of infant body keypoints using video frames as data input.
  2. Prediction models for early prediction of CP using the infant body keypoint coordinates from (1) as data inputÂ
