Deep In Motion : 17.3 Description datasets

Created by Felix Tempel on 23.06.2025

Data sources:

  • Total infants: N = 557

  • Age range: 9–18 weeks corrected age

  • Four geographic locations: Belgium, India, Norway, USA

Splits:

  • Development set (75%): used for NAS training and CV

  • Test set (25%): held out entirely during training; used only once for final performance

Splitting procedure:

  • Stratified by outcome class (CP, UL-CP, BL-CP) and site see (Groos et al.).
