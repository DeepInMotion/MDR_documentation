Deep In Motion : 16.3 Feature separation

Created by Felix Tempel, last updated by Inga Strümke on 20.08.2025

Requirements

  •  The manufacturer has not only evaluated and documented the quality measures for the different models globally, but also separately for different features, if applicable.

Summary

Test performance metrics were computed on the full test set.

Possible improvements:

  • Subgroup analysis by CP subtype (e.g., spastic bilateral vs. unilateral) and age at video capture is not yet implemented.

  • No per-feature stratified performance (e.g., performance by velocity-dominant vs. position-dominant feature windows) was reported.

Future iterations can include:

  • Demographic-based performance comparison (e.g., gestational age, term/preterm, etc.).
  • Performance variability across specific biomechanical feature categories.
